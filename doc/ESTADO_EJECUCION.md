## 📋 Convención de Registro

Cada entrada en el log sigue este formato estandarizado:

```
### [FASE X · TAREA Y] Nombre descriptivo de la tarea
- **Prompt ID:** FX-TY
- **Estado:** 🔄 Iniciada | ✅ Completada | ❌ Error | ⚠️ Con observaciones
- **Fecha/Hora:** YYYY-MM-DD HH:MM
- **Skill activo:** Ingeniero Fullstack | Diseñador UX/UI
- **Acción ejecutada:** Descripción de lo que se hizo
- **Resultado obtenido:** Qué artefacto, salida o confirmación se obtuvo
- **Archivos afectados:** lista de rutas de archivos creados o modificados
- **Observaciones:** notas, errores encontrados, decisiones tomadas
```
